# Code Folder

This folder contains the website implementation for the GreenLife eco-friendly lifestyle project.

## Files included

- `index.html` — main website page.
- `styles.css` — custom styling for the site.
- `script.js` — JavaScript and jQuery interactions.

## How to use

1. Open `index.html` in a browser.
2. Update the content to match your Figma prototype.
3. Add additional pages or sections for your full website.
4. Push this folder to the `Code` folder in the Git repository.

## Notes

- The website uses Bootstrap CDN and jQuery CDN.
- Replace placeholder text with the real project content.
- Add mobile layout screens and additional pages as required by the assignment.
